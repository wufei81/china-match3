## 💡 配色使用技巧

### 60-30-10 法则

```
60% - 主背景色（中性色）
30% - 次要色（主品牌色）
10% - 强调色（CTA、高亮）
```

### 深色模式配色技巧

1. 避免纯黑（#000），用深灰（#0f172a, #111827）
2. 降低饱和度（使用 400/500 而非 600/700）
3. 提高对比度但不刺眼
4. 使用色彩层级区分重要性

| 紫   | 奢华、创意 | 美妆、艺术、奢侈品   |
| 橙   | 活力、友好 | 娱乐、餐饮、年轻品牌 |
