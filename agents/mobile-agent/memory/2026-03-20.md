### 关键学习

1. Playwright 选择器必须使用标准 API (`getByText`, `getByRole` 等)
2. Docker 容器 API 端点可能与文档不一致，需要实际测试验证
3. 多智能体协作效率高，8 周方案可压缩到 2-3 小时完成核心功能
4. DuckDB 适合嵌入式 OLAP 场景，列存 + 压缩内存效率高
