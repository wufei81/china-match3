## 🎯 技能概述

为项目经理智能体提供团队领导能力，包括团队建设、激励方法、冲突管理、绩效反馈。帮助从"任务分配"升级为"团队赋能"。

---

## 📊 核心功能

### 1. 团队建设

**目的**: 建立高效协作的团队

#### 团队发展阶段

| 阶段 | 特征 | 领导策略 |
|------|------|---------|
| **形成期** | 相互试探 | 明确目标、角色 |
| **风暴期** | 冲突出现 | 调解冲突、建立规范 |
| **规范期** | 规则建立 | 强化协作、授权 |
| **执行期** | 高效产出 | 支持、认可 |

---

### 2. 激励方法

**目的**: 激发团队动力

#### 激励因素

| 因素 | 方法 |
|------|------|
| **成就感** | 设定挑战性目标、及时认可 |
| **成长** | 培训机会、技能发展 |
| **自主** | 授权决策、减少微观管理 |
| **归属** | 团队活动、文化建设 |

---

### 3. 冲突管理

**目的**: 有效处理团队冲突

#### 冲突解决策略

| 策略 | 适用场景 |
|------|---------|
| **协作** | 双方利益都重要 |
| **妥协** | 时间紧迫、势均力敌 |
| **迁就** | 对方正确、关系重要 |
| **回避** | 问题琐碎、冷静期 |
| **竞争** | 紧急情况、原则问题 |

---
