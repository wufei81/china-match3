# Risk Management Skill - 风险管理技能

**版本**: 1.0  
**创建日期**: 2026-03-26  
**触发词**: "风险管理", "风险识别", "风险应对", "risk management", "risk register"

---

## 🎯 技能概述

为项目经理智能体提供风险管理能力，包括风险识别、评估、应对、监控。帮助从"被动应对"升级为"主动预防"。

---

## 📊 核心功能

### 1. 风险识别

**目的**: 系统化识别潜在风险

#### 风险类别

| 类别 | 示例 |
|------|------|
| **技术风险** | 技术难点、性能问题 |
| **进度风险** | 延期、依赖延迟 |
| **资源风险** | 人力不足、关键人离职 |
| **需求风险** | 需求变更、范围蔓延 |
| **外部风险** | 政策变化、供应商问题 |

---

### 2. 风险评估

**目的**: 量化风险影响，确定优先级

#### 风险矩阵

```
           影响 (Impact)
         低 ←─────→ 高
      ┌─────────┬─────────┐
   高 │ 中风险  │ 高风险  │
概      ├─────────┼─────────┤
率   低 │ 低风险  │ 中风险  │
(      └─────────┴─────────┘
Probability)
```

#### 风险评估表

| 风险 | 概率 (1-5) | 影响 (1-5) | 风险值 | 优先级 |
|------|----------|----------|-------|-------|
| [风险] | X | X | P×I | P0/P1/P2 |

---

### 3. 风险应对

**目的**: 制定应对策略，降低风险影响

#### 应对策略

| 策略 | 描述 | 适用场景 |
|------|------|---------|
| **规避** | 改变计划避免风险 | 高风险高概率 |
| **转移** | 转移给第三方 | 外部风险 |
| **减轻** | 降低概率或影响 | 中等风险 |
| **接受** | 不采取行动 | 低风险 |

#### 风险登记册

```markdown
## 风险登记册

| ID | 风险 | 概率 | 影响 | 应对策略 | 负责人 | 状态 |
|----|------|------|------|---------|-------|------|
| R-001 | [风险] | X | X | [策略] | [姓名] | [状态] |
```
