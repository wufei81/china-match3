## 🎯 技能概述

为项目经理智能体提供成本管理能力，包括预算编制、成本控制、挣值分析、ROI 追踪。帮助从"进度管理"升级为"成本效益管理"。

## 📊 核心功能

### 1. 预算编制

**目的**: 合理规划项目成本

#### 成本类别

| 类别 | 项目 |
|------|------|
| **人力成本** | 工资、福利、外包 |
| **设备成本** | 服务器、软件许可 |
| **运营成本** | 办公、差旅、培训 |
| **应急储备** | 风险应对预算 |

#### 预算模板

```markdown
## 项目预算

### 人力成本
| 角色 | 人数 | 月成本 | 月数 | 小计 |
|------|------|-------|------|------|
| 开发 | X | ¥X 万 | X | ¥X 万 |
| 设计 | X | ¥X 万 | X | ¥X 万 |

### 总预算
- 人力：¥X 万
- 设备：¥X 万
- 运营：¥X 万
- 应急：¥X 万
- **总计**: ¥X 万
```

---

### 2. 成本控制

**目的**: 监控实际支出，避免超支

#### 挣值分析

| 指标 | 公式 | 说明 |
|------|------|------|
| **PV** (计划价值) | - | 计划完成工作的预算 |
| **EV** (挣值) | - | 实际完成工作的预算 |
| **AC** (实际成本) | - | 实际花费 |
| **CV** (成本偏差) | EV - AC | 正=节省，负=超支 |
| **SV** (进度偏差) | EV - PV | 正=提前，负=延迟 |
