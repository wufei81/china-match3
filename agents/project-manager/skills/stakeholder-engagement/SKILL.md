**版本**: 1.0  
**创建日期**: 2026-03-26  
**触发词**: "干系人管理", "利益相关者", "沟通计划", "stakeholder", "engagement"

---

## 🎯 技能概述

为项目经理智能体提供干系人管理能力，包括干系人识别、参与度评估、沟通计划、期望管理。帮助从"任务管理"升级为"关系管理"。

---

## 📊 核心功能

### 1. 干系人识别

**目的**: 识别所有相关方，理解其影响力

#### 干系人类别

| 类别 | 示例 |
|------|------|
| **内部** | 高管、团队成员、其他部门 |
| **外部** | 客户、供应商、合作伙伴 |
| **监管** | 政府机构、行业协会 |

---

### 2. 干系人分析

**目的**: 理解干系人需求和影响力

#### 权力 - 利益矩阵

```
            权力 (Power)
         低 ←─────→ 高
      ┌─────────┬─────────┐
   高 │ 随时沟通 │ 重点管理 │
利      ├─────────┼─────────┤
益   低 │ 最小努力 │ 保持满意 │
(      └─────────┴─────────┘
Interest)
```

| 姓名 | 角色 | 权力 | 利益 | 态度 | 策略 |
|------|------|------|------|------|------|

---

### 3. 沟通计划

**目的**: 确保信息有效传递
