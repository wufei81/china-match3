## 🎯 技能使用场景

### 新项目架构设计

```
1. advanced-architecture → 架构选型和容量规划
2. security-architecture → 威胁建模和安全设计
3. system-architect → 详细架构设计
4. claude-code-teams → 多视角设计审查
```

### 架构评审

```
1. security-architecture → 安全评审
2. advanced-architecture → 技术债务评估
3. system-architect → 架构文档生成
```

### 容量规划

```
1. advanced-architecture → 容量估算和成本分析
2. system-architect → 资源拓扑设计
```
