## 🎯 技能使用场景

### 新产品/功能规划

```
1. business-analysis → 市场规模和 ROI 评估
2. user-research → 用户洞察和 JTBD 分析
3. product-strategy → 愿景和路线图
4. phy-prd-writer → PRD 撰写
5. stakeholder-management → 利益相关者对齐
```

### 需求验证

```
1. user-research → 用户访谈
2. data-driven-decision → A/B 测试设计
3. firecrawl-search → 竞品分析
```

### 项目执行

```
1. stakeholder-management → 期望管理和风险沟通
2. data-driven-decision → 指标追踪和漏斗分析
3. diagram → 流程图和架构图
```

### 战略复盘

```
1. product-strategy → 路线图回顾和调整
2. business-analysis → ROI 验证
3. data-driven-decision → 数据分析
```

## 📋 技能组合模板

### 模板 1: 新产品可行性分析

**使用技能**: `business-analysis` + `user-research` + `product-strategy`

**输出**:

- TAM/SAM/SOM 市场规模
- ROI 预测
- 用户画像和 JTBD
- 产品愿景和初步路线图

### 模板 2: 功能迭代优化

**使用技能**: `data-driven-decision` + `user-research`

**输出**:

- 漏斗分析和流失点识别
- A/B 测试方案
- 用户访谈反馈
- 优化建议

### 模板 3: 跨部门项目推进

**使用技能**: `stakeholder-management` + `phy-prd-writer`

**输出**:

- 利益相关者地图
- 沟通计划
- 风险登记册
- PRD 和 WorkPackages
