## 🎯 技能概述

为 DevOps 工程师智能体提供云安全能力，包括云安全最佳实践、合规检查、密钥管理、网络安全。

---

## 📊 核心功能

### 云安全检查清单

| 领域 | 检查项 | 工具 |
|------|-------|------|
| **身份认证** | MFA 启用、最小权限 | IAM 策略 |
| **网络安全** | 安全组、NACL、VPC | 网络审计 |
| **数据安全** | 加密存储、传输加密 | KMS、TLS |
| **密钥管理** | 无硬编码、定期轮换 | Secrets Manager |
| **日志审计** | 操作日志、访问日志 | CloudTrail |

### 合规框架

| 框架 | 适用场景 |
|------|---------|
| **SOC 2** | 数据安全 |
| **GDPR** | 隐私保护 |
| **HIPAA** | 医疗健康 |
| **PCI DSS** | 支付安全 |
