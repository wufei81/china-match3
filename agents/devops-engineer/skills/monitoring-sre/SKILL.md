## 📊 核心功能

### SLO 管理

| 服务类型 | SLO 指标 | 目标 |
|---------|---------|------|
| **API 服务** | 可用性 | 99.9% |
| **API 服务** | P99 延迟 | <500ms |
| **前端服务** | 页面加载 | <3s |
| **数据库** | 查询成功率 | 99.99% |

### 错误预算管理

```
错误预算 = (1 - SLO) × 时间窗口

示例:
- SLO: 99.9%
- 月度预算：0.1% × 30 天 = 43.2 分钟

预算消耗策略:
- >50%: 正常发布
- 25-50%: 谨慎发布
- <25%: 冻结发布
```
