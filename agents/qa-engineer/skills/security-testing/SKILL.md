## 🎯 技能概述

为 QA 工程师智能体提供安全测试能力，包括 OWASP Top 10 测试、渗透测试、漏洞扫描、安全审计。帮助从"功能测试"升级为"安全质量保障"。

---

## 📊 核心功能

### 1. OWASP Top 10 测试

**目的**: 针对 OWASP Top 10 漏洞进行系统化测试

#### 测试清单

| 漏洞 | 测试方法 | 工具 | 修复建议 |
|------|---------|------|---------|
| **A01 权限控制缺失** | 越权访问测试 | Burp Suite | RBAC、服务端校验 |
| **A02 加密机制缺失** | SSL 测试、数据泄露测试 | SSL Labs | TLS 1.3、数据加密 |
| **A03 注入攻击** | SQL 注入、命令注入测试 | SQLMap | 参数化查询 |
| **A04 不安全设计** | 威胁建模审查 | - | 安全设计模式 |
| **A05 安全配置错误** | 配置扫描 | Nmap | 安全基线 |
| **A06 易失陷组件** | 依赖扫描 | Snyk | 及时更新 |
| **A07 认证失效** | 暴力破解测试 | Hydra | MFA、限流 |
| **A08 软件完整性** | 签名验证 | - | 数字签名 |
| **A09 日志监控缺失** | 审计日志检查 | - | 安全日志 |
| **A10 SSRF** | SSRF 测试 | - | URL 白名单 |

#### 安全测试模板

```markdown
## 安全测试报告

### A01: 权限控制测试
| 测试用例 | 预期 | 实际 | 结果 |
|---------|------|------|------|
| 普通用户访问管理员接口 | 403 | 403 | ✅ |
| 用户 A 访问用户 B 数据 | 403 | 403 | ✅ |
| 未登录访问受保护接口 | 401 | 401 | ✅ |

### A03: 注入攻击测试
| 测试类型 | 测试输入 | 预期 | 结果 |
|---------|---------|------|------|
| SQL 注入 | ' OR '1'='1 | 拦截 | ✅ |
| XSS 注入 | <script>alert(1)</script> | 转义 | ✅ |
| 命令注入 | ; rm -rf / | 拦截 | ✅ |

### 漏洞汇总
| 漏洞 | 严重性 | 状态 |
|------|-------|------|
| [漏洞名称] | 高/中/低 | 已修复/待修复 |
```

### 2. 渗透测试

**目的**: 模拟黑客攻击，发现安全漏洞

#### 渗透测试流程

```
1. 信息收集
   - 域名/IP 扫描
   - 端口扫描
   - 服务识别

2. 威胁建模
   - 攻击面分析
   - 威胁识别

3. 漏洞扫描
   - 自动扫描
   - 手动验证

4. 漏洞利用
   - 验证漏洞
   - 评估影响

5. 报告输出
   - 漏洞详情
   - 修复建议
```

#### 渗透测试工具

| 工具 | 用途 | 场景 |
|------|------|------|
| **Burp Suite** | Web 代理、扫描 | Web 渗透 |
| **Nmap** | 端口扫描 | 信息收集 |
| **SQLMap** | SQL 注入 | 数据库测试 |
| **Metasploit** | 漏洞利用 | 渗透测试 |
| **Wireshark** | 网络抓包 | 网络分析 |

## 🔧 使用示例

### 示例 1: OWASP 测试

**输入**: "帮我们做 OWASP Top 10 测试"

**输出**:
```markdown
## OWASP Top 10 测试报告

### 测试结果
| 漏洞 | 测试数 | 通过 | 失败 |
|------|-------|------|------|
| A01 权限控制 | 15 | 15 | 0 |
| A03 注入攻击 | 10 | 9 | 1 ⚠️ |
| A07 认证失效 | 8 | 8 | 0 |

### 发现漏洞
**A03-001: SQL 注入风险**
- **位置**: /api/search
- **严重性**: 高
- **修复**: 改用参数化查询
```
