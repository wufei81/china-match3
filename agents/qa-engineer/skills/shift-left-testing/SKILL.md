# Shift-Left Testing Skill - 左移测试技能

**版本**: 1.0  
**创建日期**: 2026-03-26  
**触发词**: "左移测试", "测试左移", "早期测试", "shift-left testing"

---

## 🎯 技能概述

为 QA 工程师智能体提供左移测试能力，将测试活动提前到开发早期，包括需求评审、设计评审、代码评审、自动化门禁。帮助从"事后测试"升级为"事前预防"。

---

## 📊 核心功能

### 左移测试策略

| 阶段 | 测试活动 | 产出物 |
|------|---------|-------|
| **需求阶段** | 需求可测试性评审 | 测试建议 |
| **设计阶段** | 设计评审、测试设计 | 测试策略 |
| **开发阶段** | 代码评审、单元测试 | 测试代码 |
| **集成阶段** | 集成测试、E2E 测试 | 测试报告 |
| **发布阶段** | 发布验证、冒烟测试 | 发布报告 |

# CI/CD 质量门禁
