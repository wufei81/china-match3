## 🎯 技能概述

为 UI/UX 设计师智能体提供设计系统构建能力，包括设计令牌、组件库、样式指南、响应式系统。帮助从"单点设计"升级为"系统设计"。

---

## 📊 核心功能

### 1. 设计令牌 (Design Tokens)

**目的**: 将设计决策代码化，确保一致性和可维护性

#### 令牌分类

| 类别 | 用途 | 示例 |
|------|------|------|
| **颜色** | 品牌色、功能色、中性色 | `--color-primary: #1890FF` |
| **字体** | 字族、大小、行高、字重 | `--font-size-base: 14px` |
| **间距** | 统一间距尺度 | `--spacing-sm: 8px` |
| **圆角** | 组件圆角 | `--radius-md: 8px` |
| **阴影** | 层级阴影 | `--shadow-md: 0 4px 12px rgba(0,0,0,0.15)` |
| **动效** | 过渡时间、缓动函数 | `--transition-fast: 150ms` |

#### 令牌模板

```json
{
  "color": {
    "primary": {
      "50": "#E6F7FF",
      "100": "#BAE7FF",
      "500": "#1890FF",
      "600": "#096DD9"
    },
    "functional": {
      "success": "#52C41A",
      "warning": "#FAAD14",
      "error": "#FF4D4F"
    }
  },
  "spacing": {
    "xs": "4px",
    "sm": "8px",
    "md": "16px",
    "lg": "24px",
    "xl": "32px"
  },
  "typography": {
    "fontFamily": "-apple-system, BlinkMacSystemFont, 'Segoe UI'",
    "fontSize": {
      "xs": "12px",
      "sm": "14px",
      "base": "16px",
      "lg": "20px"
    }
  }
}
```

### 2. 组件库规范

**目的**: 标准化组件设计，提高复用性

#### 组件文档模板

```markdown
## 组件规范 - Button

### 概述
[组件用途、使用场景]

### 变体
| 变体 | 用途 | 示例 |
|------|------|------|
| Primary | 主要操作 | [样式] |
| Secondary | 次要操作 | [样式] |
| Danger | 危险操作 | [样式] |

### 尺寸
| 尺寸 | 高度 | 字体 | 间距 |
|------|------|------|------|
| Small | 32px | 14px | 12px |
| Medium | 40px | 16px | 16px |
| Large | 48px | 18px | 20px |

### 状态
| 状态 | 背景色 | 文字色 | 边框 |
|------|-------|-------|------|
| Default | #1890FF | #FFF | #1890FF |
| Hover | #40A9FF | #FFF | #40A9FF |
| Active | #096DD9 | #FFF | #096DD9 |
| Disabled | #F5F5F5 | #999 | #D9D9D9 |
| Loading | #1890FF | #FFF | #1890FF |

### 可访问性
- 键盘支持：Tab 聚焦，Enter/Space 触发
- ARIA 角色：button
- 对比度：≥4.5:1
```

### 3. 响应式设计系统

**目的**: 确保多设备一致体验

#### 断点系统

| 断点 | 宽度 | 适用设备 |
|------|------|---------|
| **xs** | <576px | 手机竖屏 |
| **sm** | ≥576px | 手机横屏 |
| **md** | ≥768px | 平板 |
| **lg** | ≥992px | 桌面 |
| **xl** | ≥1200px | 大桌面 |
| **xxl** | ≥1600px | 超大桌面 |

#### 响应式网格

```
12 列网格系统:
- 手机：1 列 (每列 100%)
- 平板：6 列 (每列~16.6%)
- 桌面：12 列 (每列~8.3%)

间距:
- 手机：16px
- 平板：24px
- 桌面：32px
```
