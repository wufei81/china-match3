# Design Handoff Skill - 设计交付技能

**版本**: 1.0  
**创建日期**: 2026-03-26  
**触发词**: "设计交付", "设计标注", "切图导出", "design handoff", "design spec"

---

## 🎯 技能概述

为 UI/UX 设计师智能体提供设计交付能力，包括设计标注、切图导出、开发文档、协作流程。帮助从"设计完成"升级为"开发就绪"。

---

## 📊 核心功能

### 1. 设计标注规范

**目的**: 清晰传达设计意图给开发

#### 标注内容

```markdown
## 设计标注清单

### 布局标注
- [ ] 尺寸 (宽/高)
- [ ] 间距 (内外边距)
- [ ] 对齐方式
- [ ] 网格系统

### 样式标注
- [ ] 颜色值 (HEX/RGB)
- [ ] 字体 (字族/大小/行高)
- [ ] 圆角值
- [ ] 阴影参数

### 交互标注
- [ ] 状态说明
- [ ] 动效参数
- [ ] 响应式规则
```

---

### 2. 切图导出规范

**目的**: 提供开发所需的所有资源

#### 导出清单

| 资源类型 | 格式 | 倍率 | 用途 |
|---------|------|------|------|
| 图标 | SVG/PNG | 1x/2x/3x | 界面图标 |
| 图片 | WebP/JPG | 1x/2x | 内容图片 |
| 动效 | Lottie | - | 复杂动画 |

---

### 3. 开发文档

**目的**: 帮助开发理解设计

#### 文档模板

```markdown
## 设计交付文档

### 变更概述
[本次设计变更说明]

### 设计令牌
[更新的令牌列表]

### 组件清单
| 组件 | 状态 | 备注 |
|------|------|------|
| [组件] | 新增/更新 | [说明] |

### 开发注意事项
- [注意 1]
- [注意 2]

### 资源链接
- Figma: [链接]
- 切图：[链接]
```
