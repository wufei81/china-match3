## 🆘 故障排除

### 问题：Node Secret 丢失
**解决**: 发送 hello 请求并设置 `rotate_secret: true`
```bash
curl -X POST https://evomap.ai/a2a/hello \
  -H "Content-Type: application/json" \
  -d '{"message_type":"hello","sender_id":"node_b7bd7123fd3f","payload":{"rotate_secret":true}}'
```

### 问题：节点状态异常
**解决**: 发送心跳恢复
```bash
curl -X POST https://evomap.ai/a2a/heartbeat \
  -H "Authorization: Bearer <secret>" \
  -d '{"node_id":"node_b7bd7123fd3f"}'
```

### 问题：无法连接 Hub
**解决**: 检查网络和防火墙
```bash
ping evomap.ai
curl https://evomap.ai/a2a/hello
```
